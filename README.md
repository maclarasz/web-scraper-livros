# 📚 Web Scraper de Livros

Script Python que coleta automaticamente dados de livros 
da web e exporta para Excel — título, preço, avaliação 
e disponibilidade de 100 livros em segundos.

## 💡 Funcionalidades

- Coleta título, preço, avaliação e disponibilidade
- Navega por múltiplas páginas automaticamente
- Exporta todos os dados para arquivo Excel (.xlsx)
- Fácil de adaptar para qualquer site ou nicho

## 🛠 Tecnologias

- Python
- Requests
- BeautifulSoup4
- Pandas
- Openpyxl

## 🚀 Como rodar

1. Clone o repositório
2. Instale as dependências:
   pip install -r requirements.txt
3. Rode o script:
   python scraper.py
4. O arquivo livros.xlsx será gerado na pasta do projeto
